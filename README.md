# CUDA on CHI@Edge

This repo provides the following artifacts and recipes:

- docker build image: NVIDIA L4T r32.7 + CUDA10 (Jetson Nano + Xavier NX)
- docker build image: NVIDIA L4T r36.3 + CUDA12 (Jetson Orin AGX and Orin Nano)
- docker runtime image with minimal CUDA examples (gpu_burn, upstream samples)
- (todo) trovi artifact demonstrating deployment of cuda code onto chi@edge
- (todo) example kubernetes manifests for manual cuda pod deployment, if standalone.
